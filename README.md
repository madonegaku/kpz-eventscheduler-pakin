# kpz-eventscheduler-pakin / Pakin Event Scheduler

## Опис
Pakin Event Scheduler — це сучасний веб-застосунок для автоматизації планування та управління подіями. Система дозволяє користувачам створювати заходи, відстежувати часові слоти та ефективно організовувати графік зустрічей, мінімізуючи ризики накладання подій.
## Технології
- **Мова програмування:** Python 3.12
- **Фреймворк:** FastAPI (забезпечує високу продуктивність та автоматичну документацію Swagger/OpenAPI)
- **База даних:** PostgreSQL (рекомендовано для стабільної роботи з реляційними даними)
- **ORM:** SQLAlchemy або Tortoise ORM (для зручної взаємодії з БД)

## Ролі користувачів для Event Scheduler:
### Гість (Guest): неавторизований відвідувач, який може переглядати цільову сторінку (landing page), ознайомлюватися з функціоналом сервісу або переглядати обмежену інформацію про подію за прямим публічним посиланням.
### Зареєстрований користувач (User): власник акаунту, який має повний доступ до свого календаря. Він може створювати власні події, редагувати їх, налаштовувати персональні нагадування та бачити свій розклад.
### Організатор (Organizer): це статус зареєстрованого користувача в контексті конкретної події. Він має право надсилати запрошення іншим людям, керувати списком учасників та змінювати деталі заходу, які бачать усі запрошені.
### Учасник / Запрошений (Attendee): користувач, який отримав запрошення на подію. Він може підтвердити або відхилити участь, переглядати деталі заходу та отримувати автоматичні сповіщення від системи.
### Адміністратор (Admin): технічний фахівець з повним доступом до системи. Він керує базою користувачів, проводить модерацію контенту, стежить за роботою сервера сповіщень та вирішує конфліктні ситуації.
